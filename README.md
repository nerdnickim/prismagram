# prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [] Like / UnLike a photo
- [] comment on a photo
- [] Search by user
- [] See user profile
- [] Follow / Unfollow User
- [] See the full photo
- [] Edit my profile
- [] Upload a photo
- [] Edit the photo (Delete)
- [] See the feed
