# prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [x] Like / UnLike a photo
- [x] comment on a photo
- [x] Search by user
- [x] See user profile
- [x] Follow User
- [x] Unfollow User
- [x] Edit my profile
- [x] See User profile
- [x] See My profile
- [x] See the full photo
- [] Upload a photo
- [] Edit the photo (Delete)
- [] See the feed
